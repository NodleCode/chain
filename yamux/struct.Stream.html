Struct yamux::Stream[−][src]

pub struct Stream { /* fields omitted */ }
Expand description

A multiplexed Yamux stream.

Streams are created either outbound via crate::Control::open_stream or inbound via crate::Connection::next_stream.

Stream implements AsyncRead and AsyncWrite and also futures::stream::Stream.
