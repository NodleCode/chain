pub struct BoundedVec<T, S>(_, _);
Expand description

A bounded vector.

It has implementations for efficient append and length decoding, as with a normal Vec<_>, once put into storage as a raw value, map or double-map.

As the name suggests, the length of the queue is always bounded. All internal operations ensure this bound is respected.

pub fn as_ptr(&self) -> *const T

Returns a raw pointer to the vectorâ€™s buffer.

The caller must ensure that the vector outlives the pointer this function returns, or else it will end up pointing to garbage. Modifying the vector may cause its buffer to be reallocated, which would also make any pointers to it invalid.

The caller must also ensure that the memory the pointer (non-transitively) points to is never written to (except inside an UnsafeCell) using this pointer or any pointer derived from it. If you need to mutate the contents of the slice, use as_mut_ptr.

Examples

let x = vec![1, 2, 4];
let x_ptr = x.as_ptr();

unsafe {
    for i in 0..x.len() {
        assert_eq!(*x_ptr.add(i), 1 << i);
    }
}
