Expand description

The name of a runtime library routine.

Runtime library calls are generated for Cranelift IR instructions that donâ€™t have an equivalent ISA instruction or an easy macro expansion. A LibCall is used as a well-known name to refer to the runtime library routine. This way, Cranelift doesnâ€™t have to know about the naming convention in the embedding VMâ€™s runtime library.
