Migrate the entire storage of this pallet to a new prefix.

This new prefix must be the same as the one set in construct_runtime. For safety, use PalletInfo to get it, as: <Runtime as frame_system::Config>::PalletInfo::name::<ElectionsPhragmenPallet>.

The old storage prefix, PhragmenElection is hardcoded in the migration code.
