Expand description

Unroll the given for loop

Example:

ⓘ
unroll! {
  for i in 0..5 {
    println!("Iteration {}", i);
  }
}

will expand into:

ⓘ
{ println!("Iteration {}", 0); }
{ println!("Iteration {}", 1); }
{ println!("Iteration {}", 2); }
{ println!("Iteration {}", 3); }
{ println!("Iteration {}", 4); }
